 
 ### Array

> 数组 2 种创建方式
* var array = new Array('red', 'blue', true, 100)
* var array = [1, 2, true, {}]

> 数组的常用方法

| 方法    | 类别 | 返回值      | 描述                                                         | 版本 |
| :------ | ---- | ----------- | ------------------------------------------------------------ | ---- |
| isArray | 类   | true，false | 检测是否为数组                                               | 2015 |
| from    | 类   | []          | 浅拷贝的数组实例                                             | 2015 |
| of      | 类   | []          | 创建一个具有可变数量参数的新数组实例，而不考虑参数的数量或类型 | 2015 |
