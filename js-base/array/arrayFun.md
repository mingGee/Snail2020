 
 ### Array

> 数组 2 种创建方式
* var array = new Array('red', 'blue', true, 100)
* var array = [1, 2, true, {}]

> 数组的常用方法

|  方法名   | 方法类别 | 返回值 |是否改变原数组  |  描述  | ECMA版本 |
|  ----  | ----  | ---- | ---- | ---- |
| isArray | 类方法 |  true or false | 否 | 检测某个值是不是数组 | 2015 |
| from | 类方法 | 数组 | 否 | 创建一个新的，浅拷贝的数组实例 | 2015 |
